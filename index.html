About Me

I’m a front-end developer focused on building clean, responsive interfaces with HTML, CSS, and JavaScript. Through Coursera’s Front-End Development program I designed and built the Mangata & Gallo e-commerce portfolio site from scratch, simulating a real product experience from homepage to checkout.

Before front-end, I worked in IT support, which taught me to stay calm under pressure, troubleshoot methodically, and take responsibility for the user experience from the moment they report a problem until it’s solved. That background shapes how I build interfaces today: I care about clarity, reliability, and making things feel simple for the person using them.

On the technical side, I work with semantic HTML, modern CSS (Flexbox, Grid, custom properties), and JavaScript to add interactivity and basic state management. I pay attention to layout, spacing, and typography, but I also try to keep my code organized and readable so it’s easy to extend and maintain.

Right now I’m looking for opportunities as a junior front-end developer where I can contribute to real projects, learn from more experienced engineers, and keep growing my skills in responsive design, accessibility, and component-based UI.
